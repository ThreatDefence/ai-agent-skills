# AI Agent Skills

A collection of skills for AI Agents. Skills are packaged instructions and scripts that extend agent capabilities.

## Available Skills

---

### threatdefence-alert-interactions

Knowledge on how to query and interact with the ThreatDefence platform's alerts API, allowing analysts to analyse, summarise, and handle alerts.

#### Requirements:

An API key with access to Alert Center, available from the [Customer Portal](https://portal.threatdefence.io/my-tenants/api-keys)

#### Use when:

- Performing general triage
- Generating reports
- Analysing trends
